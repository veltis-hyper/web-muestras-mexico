# ✦ Web Muestras México v2.0

> **30 páginas web de demostración para negocios mexicanos**  
> Diseño psicológicamente optimizado · Front-end impecable · Imposibles de rechazar

---

## 🎯 ¿Qué es esto?

Este es un **sistema de ventas de páginas web** diseñado para vender a negocios locales mexicanos que aún no tienen presencia digital. Contiene:

- **30 demos web profesionales** — una para cada empresa real encontrada en México
- **Landing maestra** con todas las empresas en un solo lugar
- **Materiales de ventas** — scripts, emails, contrato y guía de ventas
- **Diseño psicológico** — CTA estratégicos, prueba social, urgencia, reciprocidad

---

## 🧠 El Método del Secuestro de Atención

Un sistema de 4 pasos para vender páginas web sin que el cliente pueda rechazarte:

| Paso | Nombre | Psicología | Acción |
|------|--------|-----------|--------|
| 1 | **El Regalo** | Reciprocidad | Entregas la demo sin pedir nada |
| 2 | **El Gancho** | Compromiso | Llamas 24-48h después para "saber qué le pareció" |
| 3 | **El Cierre** | Escasez + Autoridad | Precio especial de introducción: $5,500 MXN |
| 4 | **La Escala** | Prueba social | Pides testimonios y referencias |

## 🛠 Cómo usar este sistema

### 1. Contactar a la empresa
Usa el script personalizado en `scripts_y_emails/script_[empresa].md`. WhatsApp es el canal más efectivo para negocios pequeños en México.

### 2. Enviar la demo
Envía el enlace directo a la demo de su empresa:  
`https://veltis-hyper.github.io/web-muestras-mexico/empresas/[slug]/index.html`

### 3. Seguimiento 24-48h
Llama o envía mensaje preguntando qué le pareció. No vendes — escuchas.

### 4. Cierre
Presenta la oferta con urgencia. Usa el contrato impreso. Toma el 50% de anticipo.

## 🏗 Cómo desplegar

### Opción 1: GitHub Pages (gratis)
1. Fork o clona este repo
2. Ve a Settings → Pages → Source: Deploy from a branch → `main` / `root`
3. Tu sitio estará en `https://[tu-usuario].github.io/web-muestras-mexico`

### Opción 2: Netlify Drop (gratis, más rápido)
1. Ve a [netlify.com/drop](https://netlify.com/drop)
2. Arrastra la carpeta `web_ventas_v2/` completa
3. Obtén una URL instantánea

### Opción 3: Cloudflare Pages (gratis, CDN global)
1. Conecta este repo a Cloudflare Pages
2. Configura build: `None` (sitio estático)
3. Deploy automático en cada push
